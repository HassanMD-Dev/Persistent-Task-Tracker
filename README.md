# Persistent Task Tracker

A simple and responsive **Task Tracker (Todo App)** built with **HTML, CSS, and JavaScript**.  
This application allows users to add, store, and delete tasks. Tasks remain saved even after refreshing the page using the browser's **LocalStorage**.

---

## Features

- Add new tasks
- Delete tasks
- Persistent task storage using LocalStorage
- Clean and modern user interface
- Lightweight and easy to use

---

## Technologies Used

- **HTML5** – Structure of the application  
- **CSS3** – Styling and layout  
- **JavaScript (ES6)** – Application logic and DOM manipulation  
- **LocalStorage API** – Persistent data storage

---

## How It Works

1. Enter a task in the input field.
2. Click the **Add** button to create a new task.
3. The task is saved in the browser using **LocalStorage**.
4. When the page reloads, saved tasks are automatically displayed.
5. Click the **Delete** button to remove a task.
